**Delete photo**
----
  Returns json data about a deleted user.

* **URL**

  /api/v1/users/:id

* **Method:**

  `DELETE`
  
*  **URL Params**
  
    | Field name   |      Type      |  Description | Require |
    |:------------:|:--------------:|:------------:|:-------:|
    |id            |Integer         |              | true    |


* **Data Params**

  None
  
* **Success Response:**

  * **Code:** `200`  <br />
    **Content:** `{message: "The user was successfully removed"}`
     
* **Error Response:**
    
   * **Code:** `404` NOT FOUND <br />
       **Content:** `{ error : "This user doesn't exist" }`

* **Sample Call:**

  ```javascript
    $.ajax({
      url: "/api/v1/users/1",
      dataType: "json",
      type : "DELETE",
      success : function(r) {
        console.log(r);
      }
    });
  ```
